CDISC for AI

Machine-actionable reference files built from CDISC SDTM Controlled Terminology, NCIt, and COSMoS Biomedical Concepts and Dataset Specializations. The purpose is to make existing standards more consumable for AI, rule engines, and humans.

Source repository: github.com/kerfors/cdisc-for-ai

Reference files

  • SDTM_Test_Identity.xlsx domain-level TESTCDs with NCIt identity
  • COSMoS_BC_DSS.xlsx flattened COSMoS Biomedical Concepts and Dataset Specializations
  • Specimen_Findings.xlsx specimen-based Findings consumer view (LB, MB, MI, CP, BS, MS, PC, PP)
  • Measurement_Findings.xlsx measurement-based Findings consumer view (VS, MK, CV)

COA structure — 6MWT walkthroughs

  • 6MWT — The NCIt Story 6-Minute Walk Test traced through NCI Thesaurus identity
  • 6MWT — The COSMoS Story 6-Minute Walk Test traced through COSMoS Biomedical Concepts and Dataset Specializations
