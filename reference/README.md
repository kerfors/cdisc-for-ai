# Reference materials

Background documents, articles, and specs used in development of cdisc-for-ai.

This folder is **gitignored** — contents are local only. Third-party documents (articles, specs) are not distributed with the repo to avoid copyright and size concerns.

## How to use

Place PDFs, articles, and supporting documents here. Claude reads them on demand when working in this repo. Organize by topic:

```
reference/
├── articles/       ← Published papers, blog posts, presentations
├── specs/          ← CDISC specs, SDTMIG excerpts, COSMoS documentation
└── notes/          ← Your own notes and summaries (could be tracked if desired)
```

## What belongs here

- Articles informing design decisions (FAIR principles, linked data, etc.)
- CDISC specification excerpts relevant to the analytical work
- COSMoS documentation and design rationale
- Standards governance and architecture papers
- Your own analysis notes and working documents
